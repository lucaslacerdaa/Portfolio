Introduction

Understand more about the project

This project came from Felipe Santana's Figma Course, aka Feux (@feuxdesign), which aims to create an app for NFT trading in cryptocurrencies. A subject that is trending in the market. In this project, we start from wireframe development to high-fidelity prototyping. Exploring the most of Figma.
It is important to remember that this project is a case study, with no financial commitment.

Iconography

The icons used in this project follow a well-defined line, where each one was methodically explained so that the student can have the autonomy to make other icons as they deem necessary. So the entire process is done solely in Figma and only the images used were made by an external creator.

Conclusão

Learning to master this tool added value to my career as Product Design, as it raised my technique to levels that I didn't even know were possible. I learned things like high-level componentization, as well as component encapsulation, microinteractions, I broke down the entire Auto Layout, Style Guide among many other things. I would like to thank my coworkers who encouraged me, especially Luiza @luizaanandamac, to continue the project and to Feux Design who conveys everything with fabulous teaching.

Thank you for getting here. 😁
